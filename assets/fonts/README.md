- Large, clear font for questions and answers

- Examples: Poppins, Quicksand, Varela Round

## Current Status
Currently using Godot's default font. Custom fonts can be imported and applied via theme overrides in the scene files.

## How to Add Custom Fonts
1. Download TTF or OTF font files
2. Place them in this directory
3. Import them in Godot
4. Apply via Theme overrides in scene nodes or create a global theme
